# Gameplay & Progression

## Core Gameplay Loop
1. Explore interconnected rooms
2. Collect progression-critical components
3. Reach the Control Center
4. Trigger the final encounter

## Progression Model
- Progression is inventory-driven, not sequence-driven
- All required components must be collected
- Prevents sequence breaking and premature completion
